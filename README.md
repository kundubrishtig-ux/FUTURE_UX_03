# Task 3 – CRM / Client Management Dashboard UI

Business Type: Digital Marketing Agency CRM

## Screens Designed

* Main Dashboard
* Lead Pipeline
* Client List / Client Profile
* Task & Follow-up Tracker

## UX Design Rationale

The CRM dashboard is designed to help small agencies efficiently manage leads and clients. The main dashboard provides a quick overview of key metrics such as total leads and active clients. The lead pipeline visualizes the sales process using status-based stages to help teams track progress.

The client list screen organizes customer information for easy access, while the task tracker ensures team members stay on top of follow-ups and important actions. The interface prioritizes clarity, workflow efficiency, and simple navigation to support real-world agency operations.
